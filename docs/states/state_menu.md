# Modulo: state_menu

## Descrizione
Semplice menu principale renderizzato tramite `ui_renderer`.

## Funzioni
### `menu_init`
- Inizializza il renderer UI.
- Mostra il cursore del mouse.

### `menu_draw`
- Disegna sfondo, titolo e pulsante "PLAY".

### `menu_update`
- Attende pressione di ENTER o click sul pulsante per passare a `STATE_GAMEPLAY`.
